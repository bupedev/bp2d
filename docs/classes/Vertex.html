Static PRECISION_THRESHOLD

PRECISION_THRESHOLD: number = 1E-9
  • Defined in geometry/vertex.ts:12

The precision threshold for comparing vertices. If the difference in axis coordinates between two vertices are both less than this value, they are considered to be equivalent. This is to mitigate small error margins that are to be expected for floating point operations.
